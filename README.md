# Hydra

Easy-to-use light issue manager.

## Recommended Stack

- [electron-vite](https://electron-vite.org/) + [React](https://react.dev/) + [Chakra UI](https://www.chakra-ui.com/)

## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
